fetch_latest_temperature <- function(station_info) {
  param <- "sea_water_temperature"
  url <- sprintf(
    "https://erddap.secoora.org/erddap/tabledap/gov-nps-ever-%s.csv?time%%2Clatitude%%2Clongitude%%2C%s&time%%3E=%s&time%%3C=%s",
    station_info$name,
    param,
    start_time,
    end_time
  )

  resp <- request(url) |>
    req_perform()

  df <- read_csv(resp_body_string(resp), show_col_types = FALSE)
  names(df) <- c("time", "latitude", "longitude", "temperature")
  df <- df[-1, ] |>
    mutate(
      time = as.POSIXct(time, format = "%Y-%m-%dT%H:%M:%SZ", tz = "UTC"),
      latitude = as.numeric(latitude),
      longitude = as.numeric(longitude),
      temperature = as.numeric(temperature)
    ) |>
    filter(!is.na(temperature))

  if (nrow(df) == 0) {
    return(NULL)
  }

  latest <- df[which.max(df$time), ]
  tibble(
    station_id = station_info$name,
    station = station_info$long_name,
    location = sprintf(
      "%s (%s)",
      station_info$long_name,
      sub("f1$", "", station_info$name)
    ),
    time = latest$time,
    latitude = latest$latitude,
    longitude = latest$longitude,
    temperature = latest$temperature
  )
}

temperature_data <- bind_rows(lapply(ndbc_stations, fetch_latest_temperature))

Temperature is shown in degrees Celsius. Cooler values (blue) and warmer values (red) reflect spatial variation across the buoy network.

Interpolated temperature field

The field below uses DIVAnd (Data-Interpolating Variational Analysis) with bathymetry masking, following the same approach as sfer-mbon-oxygen: ~0.01° horizontal grid resolution (~1 km), correlation length (0.05, 0.05) degrees, and epsilon2 = 1.0. Land is masked using Natural Earth 10 m coastlines plus NOAA BlueTopo 4 m bathymetry (8 m tiles fill gaps); cells with missing bathymetry or elevation ≥ −1 m are treated as land.
